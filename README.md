<h3>How to use</h3>
<ul>
    <li><code>yarn install</code> : Install dependencies</li>
    <li><code>yarn dev</code> : Start development server</li>
    <li><code>yarn build</code> : Build the project</li>
    <li><code>yarn start</code> : Start production server</li>
</ul>
